# supRA
A WIP customizable ICMPv6 Router Advertisement (RA) sender.

[lua 5.4](https://www.lua.org/ftp/lua-5.4.4.tar.gz) is required.

## Config
See [config.sample.lua](./config.sample.lua) for more information.
